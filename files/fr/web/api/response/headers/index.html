{{APIRef("Fetch")}}

La propriété en lecture seule headers de l'API {{domxref("Response")}} contient l'objet {{domxref("Headers")}} associé à la réponse.

Synthaxe

var myHeaders = response.headers;

Valeur

Un objet {{domxref("Headers")}}.

Exemple

Dans notre exemple (voir la version en ligne) nous avons créé un nouvel objet {{domxref("Request")}} en utilisant le constructeur {{domxref("Request.Request","Request()")}} avec un chemin vers un fichier JPG en argument. On récupère (fetch en anglais) ensuite la requête avec {{domxref("GlobalFetch.fetch","fetch()")}}, on extrait un blob de la réponse grâce à {{domxref("Body.blob")}} pour créer un objet URL via la méthode {{domxref("URL.createObjectURL")}} et l'afficher dans une balise {{htmlelement("img")}}.

Notez que nous affichons les valeurs des headers de la réponse dans la console en haut du bloc de code fetch()

var myImage = document.querySelector('img');

var myRequest = new Request('flowers.jpg');

fetch(myRequest).then(function(response) {
  console.log(response.headers); // retourne un objet Headers{} 
  response.blob().then(function(myBlob) {
    var objectURL = URL.createObjectURL(myBlob);
    myImage.src = objectURL;
  });
});
