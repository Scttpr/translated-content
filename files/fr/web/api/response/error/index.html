La méthode error() de l'API {{domxref("Response")}} retourne un nouvel objet Réponse associé à une erreur réseau.

Note : ceci est surtout pertinent dans le cas des ServiceWorkers; la méthode est utilisée pour retourner une erreur si vous le souhaitez. Une réponse d'erreur a son {{domxref("Response.type","type")}} défini en error.

Note: Une Response d'erreur n'est jamais vraiment exposée au script : une telle réponse à un {{domxref("GlobalFetch.fetch","fetch()")}} rejetterait la promesse.

Synthaxe

var errorResponse = Response.error();

Paramètres

Aucun.

Valeur retournée

Un objet {{domxref("Response")}}.
